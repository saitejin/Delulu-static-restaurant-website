# Delulu Restaurant Website

> A static restaurant website built with HTML and CSS featuring a modern, responsive design.

---

## 🍽️ Overview

Delulu is a static restaurant website designed to showcase the restaurant's menu, ambiance, and contact information. The site features a clean dark theme with orange accents, smooth animations, and a fully responsive layout for all screen sizes.

---

## 📄 Pages

| Page | Description |
|---|---|
| Home | Hero section, latest dishes, desserts, and why choose us |
| About | Restaurant story and ambiance gallery |
| Menu | Full food and dessert menu with prices |
| Contact | Contact form, location map, and social links |

---

## ✨ Features

- Responsive design for mobile, tablet, and desktop
- Animated typewriter effect on hero heading
- Hamburger menu for mobile navigation
- Food and dessert cards with hover effects
- Contact form with map integration
- Fixed navbar with blur effect
- Social media links in footer

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Page structure |
| CSS3 | Styling and animations |
| Bootstrap Icons | Icons throughout the site |
